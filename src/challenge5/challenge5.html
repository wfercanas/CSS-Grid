Challenge 5

Create two grids where:

  1. Both grids use all the available space for its width.
  2. The first grid has 4 items of the same size and they grow their width as much as it be required to fill all the available space.
  3. The second grid has 4 items of the same size and even though the grid spans all the available space, as the grid width grows, new slots should be created of the same size of the already defined ones.
  4. All items from both grids should always be greater than 100px.

Resize the screen and turn on the grids in the Inspector.

Check auto-fit and auto-fill keywords.
